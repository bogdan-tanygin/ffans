Basic manual:

- Button "S" to see nanoparticles visualized as spheres;
- Button "M" to see magnetic moment directions;
- Space to start/stop simulation;
- Mouse on-click drag to rotate camera;
- Ctrl + mouse to zoom in/out

Other details of interactivity, input/output and parameters (including number of particles) can be found in the source code.
